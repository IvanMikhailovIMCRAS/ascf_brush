$` u(z) = V(z) - V(H) = \beta \cdot \ln(\cos(K \cdot z)/ \cos(K \cdot H))`$,

$` u(z) = -\ln[1-\varphi(z)]`$,

$` \varphi(z) = \exp[-u(z)]`$,

$` \varphi(z)= \left[\frac{\cos(K \cdot H)}{\cos(K \cdot z)} \right]^\beta`$
